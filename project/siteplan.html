TastyBites – Restaurant Site Plan

Site Name

TastyBites – A catchy, memorable name for a restraurant that reflects delicious food and a modern vibe.

Site Purpose

The site showcases the restaurant's offerings, with a browsable menu, takeaway order form, and feature to save favorite dishes. It's designed to be responsive and user-friendly for both desktop and mobile users.

Scenarios

  • “I want to quickly check what dishes are available before visiting.”
  • “Can I place a takeaway order directly from my phone?”

Color Schema

  • Primary: #C0392B (Rich Red)
  • Secondary: #F4F1EA (Warm Off-white)
  • Accent: #F5B041 (Gold/Yellow)
  • Text: #2C3E50 (Dark Gray-Blue)

Typography

  • Headings: Playfair Display (Serif)
  • Body: Open Sans (Sans-serif)
